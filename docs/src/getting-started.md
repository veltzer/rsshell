# Getting Started

## Running the shell

```bash
# Start interactive shell
rsshell

# Run a single command
rsshell -c "echo hello world"

# Initialize config file
rsshell init-config

# Show version info
rsshell version
```

## First steps

1. Run `rsshell init-config` to create a default config at `~/.config/rsshell/config.toml`
2. Edit the config to set up aliases and environment variables
3. Run `rsshell` to start the interactive shell
