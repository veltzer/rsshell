# Installation

## Install from crates.io

```bash
cargo install rsshell
```

This downloads, compiles, and installs the latest published version into `~/.cargo/bin/`.

## From source

```bash
git clone https://github.com/veltzer/rsshell.git
cd rsshell
cargo build --release
```

The binary will be at `target/release/rsshell`.

## From GitHub Releases

Download pre-built binaries from the [Releases page](https://github.com/veltzer/rsshell/releases). Available for Linux (x86_64, aarch64), macOS (x86_64, aarch64), and Windows (x86_64).
